Transformación entre formas de funciones cuadráticas

Las tres formas principales de una función cuadrática son:

  • Cartesiana: f(x) = ax² + bx + c
  • Factorizada: f(x) = a(x - x1)(x - x2)
  • Canónica: f(x) = a(x - h)² + k
De cartesiana a factorizada: Se aplican las fórmulas para encontrar las raíces x1 y x2: f(x) = ax² + bx + c x1, x2 = (-b ± sqrt(b² - 4ac)) / 2a f(x) = a(x - x1)(x - x2)De cartesiana a canónica: f(x) = ax² + bx + c h = -b / 2a k = f(h) = a(h)² + b(h) + c f(x) = a(x - h)² + k De canónica a cartesiana: f(x) = a(x - h)² + k f(x) = a(x² - 2hx + h²) + k f(x) = ax² - 2ahx + ah² + k De factorizada a cartesiana: f(x) = a(x - x1)(x - x2) f(x) = a(x² - (x1 + x2)x + x1x2)

Ejercicio 2: Convertir f(x) = 2x² + 4x - 6 a forma factorizada y canónica.

Ejercicio 4: Expandir f(x) = 2(x - 3)² - 4 a forma cartesiana.
Ejercicio 5: Convertir f(x) = -x² + 2x + 3 a forma factorizada y canónica.
